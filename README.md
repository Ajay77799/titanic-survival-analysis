# Project 6 - Titanic Survival Analysis

## Overview
Activity-based Learning Model — Healthcare Data Insights pattern applied to the Titanic dataset.

## Tasks
- Task A: Survival rate by Sex
- Task B: Survival rate by Age Group  
- Task C: Survival rate by Passenger Class
- ML Prediction: Random Forest (81.56%) and Logistic Regression (79.89%)

## Dataset
https://www.kaggle.com/c/titanic/data

## How to Run
```bash
pip install -r requirements.txt
python3 titanic_analysis.py
```

## Results
- Random Forest Accuracy: 81.56%
- Logistic Regression Accuracy: 79.89%
- Top features: Sex, Fare, Age
